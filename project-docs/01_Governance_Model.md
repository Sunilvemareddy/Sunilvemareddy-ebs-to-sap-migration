# Governance Model

## Governance Bodies
- **Executive Sponsor**
- **Steering Committee** — monthly; approves scope, budget, escalations
- **Program Management Office (PMO)** — runbooks, schedule, reporting
- **Workstream Leads** — Data, Functional, Technical, Testing, Change

## Decision authority
- Sponsor -> Steering -> Program Manager -> Workstream Leads -> SMEs

## Meeting cadences
- Steering Committee: Monthly
- Program status: Weekly (PMO)
- Workstream standups: Daily or tri-weekly (as required)

## Escalation matrix
| Level | Role | SLA |
|---|---:|---:|
| 1 | Workstream Lead | 24 hrs |
| 2 | Program Manager | 48 hrs |
| 3 | Steering Committee | 72 hrs |
