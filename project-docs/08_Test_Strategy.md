# Test Strategy (Overview)

## Test Types
- Unit tests (component)
- Integration tests (interfaces)
- End-to-end business scenarios (UAT)
- Performance & load testing
- Data reconciliation testing

## Assets
- Test scripts & scripts traceability matrix
- Test data (masked production subset)
- Defect tracking via Jira (link)

## Entry/Exit criteria
Define for each test cycle and require business sign-off for UAT exit.
